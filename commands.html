Использование команд

[ параметр ] - опциональный параметр

< параметр> - обязательный параметр

< параметр1 | параметр2> - параметр на выбор

. - префикс команд по умолчанию

- параметр настройки

help
Использование
help [ число ]

Отображает по страницам все доступные команды на сервере

Пример
help 5 help 6
Баги

Не отбражает все команды

В разделах между командой и описанием нет пробела

В разделах на ПК в названии появляется черная полоска в начале

Слово котегорий когда правильно должно быть категорий

ХЗ

Мне сказали сделать этот блок

serverinfo
Использование
serverinfo

Выводит информацию о создателе, количеству участников и доступные емодзи

Пример
serverinfo
Баги

Не всегда отображает создателя сервера

Количество ролей на 1 меньше чем команада отображает

botinfo
Использование
botinfo

Выводит информацию пинге, количеству команд и серверов, версию, приглашение и время работы

Пример
botinfo
Баги

Не пишет текущее время в низу сообщения, как в остальных командах

ping
Использование
ping

Выводит пинг сервера бота и апи дискорда

Пример
ping
Баги

Иногда не может найти пинг сервера бота

invite
Использование
invite

Отбражает ссылку на приглошение бота и на сервер поддержки

Пример
invite
Баги

Ссылки на приглошение бота и на сервер перепутаны

uptime
Использование
uptime

Отбражает время непрерывной работы бота

Пример
uptime
Баги

Во вкладке всего текст на английском

clickerlink
Использование
clickerlink

Отбражает описание сайтовой игры кликер

Пример
clickerlink
Баги

Ссылка на источник не работает

ban
Использование
ban < @участник > [ причина ]

Банит участника или бота с сервера

Пример
ban @cat ты ПлАхОй ban @dog
Баги

Можно забанить самого себя, но бот нечего не сделает

Бот может попытаться забанить самого себя, но нечего не сделает

Если нет у бота нужного разререшения, он нечего не сделает

Через бота нельзя указать причину бана

unban
Использование
unban < @участник >

Возвращает участника с списка банов сервера

Пример
uban @cat
Баги

Этой комманды не существует

warn
Использование
warn < @участник>

Дает предупреждение участнику. После 3 нарушений участник попадает в список банов

Пример
warn @cat warn @dog ты нарушил пункт 15
Баги

Можно прописать себе варн

Если бот не может тебя выгнать при получении 3/3 варнов то он отправит пустое сообщение

warns
Использование
warns [ @участник ]

Посмотреть сколько раз тебе или давали варн

Пример
warns warns @cat
Баги

Если не указать пользователя, бот должен показать варны отправителя сообщения

unwarn
Использование
unwarn < @участник>

Убирает предупреждение участнику

Пример
unwarn @cat
Баги

Можно убрать у себя варн

clear
Использование
clear < количество>

Убирает указаное количество сообщений. Количество может быть от 1 до 100 сообщений

Пример
clear 50
Баги

Нет защиты на проверку typeof string

Нет пробела после точки

Можно указать как количество число 0

kick
Использование
kick < @участник>

Кикает участника или бота сервера

Пример
kick @cat ты плАхой kick @dog
Баги

Можно выгнать самого себя, но бот нечего не сделает

Бот может попытаться выгнать самого себя, но нечего не сделает

Если нет у бота нужного разререшения, н нечего не скажет

Через бота нельзя указать причину выгона

idea
Использование
idea < идея>

Посылает идею для бота на сервер разработчика

Пример
idea Сделай настройку префикса
bug
Использование
bug < баг>

Посылает описание бага бота на сервер разработчика

Пример
bug Черная полоска в названии раздела
recall
Использование
recall < отзыв>

Посылает отзыв о боте на сервер разработчика

Пример
recall 7/10 Неплохой бот, но еще сырой
plaint
Использование
plaint < жалоба>

Посылает жалоба о боте на сервер разработчика

Пример
plaint Бот удалил мой сервер
daily
Использование
daily

Забрать зарплату своей роли

Пример
daily
balance
Использование

Пример

Кликнуть и получить 1$ в игре кликер

Пример
click

Посмотреть свой или чужой баланс в игре кликер

Пример
